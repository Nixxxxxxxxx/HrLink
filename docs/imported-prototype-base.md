## Зачем это сделано

Этот импорт нужен как быстрый рабочий baseline:

- сохранить уже собранные create/view news сценарии;
- не потерять generated UI-структуру;
- использовать существующий app shell как стартовую точку для дальнейшей миграции в HR-продукт.
